# Campañas

Un directorio por campaña: `<YYYY-MM-slug>/`
- `BRIEF.md` — segmento, ángulo, offer, lista estimada, hipótesis, criterio de éxito (lo crea /gtm-campaign-ideation; /gtm-experiments agrega el diseño del A/B)
- `COPY.md` — secuencias por variante con framework e hipótesis de dolor (lo crea /gtm-copy)
- `RESULTS.md` — números crudos, decisión del experimento, aprendizajes (lo crean /gtm-experiments y /gtm-reply-analysis)
